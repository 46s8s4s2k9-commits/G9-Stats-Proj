# Study Method × Income → Exam Score

**A Statistical Investigation**

A statistical investigation into whether study method (Offline, Online, Hybrid) and family income level are associated with final exam scores. Uses a Kaggle dataset of 8,000 students.

### Authors

Bryan Tu, Andy Sha, Kingsley Sy

### How to view

Open `index.html` in a browser. No build step or dependencies to install.

### Data source

[Student Lifestyle vs. Academic Performance Dataset](https://www.kaggle.com/datasets/sarveshchhetri/student-lifestyle-vs-academic-performance-dataset) — Kaggle

### Tools used

Microsoft Excel for descriptive statistics and randomization simulations; Chart.js for visualizations.
